        <p class="eyebrow">The Problem</p>
        <h2>Seamless looping looks smooth. It's clinically wrong.</h2>
        <p>
          Most exercise apps use a shortcut: record half a movement, reverse it, and join the two halves. The result plays back as a smooth loop. For symmetric movements it's invisible. For any exercise with a deliberate tempo difference between phases, it silently destroys the clinical intent.
        </p>
      </div>

      <div class="problem-grid">

        <div class="problem-text fade-in">
          <p>
            Take a squat with a 3-second eccentric (lowering) phase and a 1-second concentric (rising) phase. That ratio is the prescription — slow eccentric loading is the therapeutic signal.
          </p>
          <p>
            A loop-shortcut app records only the downward phase, then reverses it to create the upward phase. This forces both halves to the same duration: 3 seconds down, 3 seconds up. The tempo ratio is gone.
          </p>
          <p>
            This matters for eccentric tendon rehab, tempo-based strength protocols, neuromuscular retraining, and any program where phase timing is the dose, not decoration.
          </p>
          <p>
            <strong>Kinetica never does this.</strong> Every video is the real movement, at the clinician-specified tempo, played back exactly as recorded.

<script>
  // Theme management
  (function() {
    const KEY = 'kinetica-theme';
    const root = document.documentElement;
    const btn  = document.getElementById('themeToggle');

    function getPreferred() {
      const stored = localStorage.getItem(KEY);
      if (stored) return stored;
      return window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
    }

    function applyTheme(t) {
      root.setAttribute('data-theme', t);
      localStorage.setItem(KEY, t);
      btn.setAttribute('aria-label', t === 'dark' ? 'Switch to light mode' : 'Switch to dark mode');
    }

    applyTheme(getPreferred());

    btn.addEventListener('click', () => {
      applyTheme(root.getAttribute('data-theme') === 'dark' ? 'light' : 'dark');
    });

    // Listen for system preference changes (if no stored choice)
    window.matchMedia('(prefers-color-scheme: dark)').addEventListener('change', e => {
      if (!localStorage.getItem(KEY)) applyTheme(e.matches ? 'dark' : 'light');
    });
  })();

  // Scroll fade-in
  (function() {
    if (window.matchMedia('(prefers-reduced-motion: reduce)').matches) return;

    const els = document.querySelectorAll('.fade-in');
    const obs = new IntersectionObserver((entries) => {
      entries.forEach(e => {
        if (e.isIntersecting) {
          e.target.classList.add('visible');
          obs.unobserve(e.target);
        }
      });
    }, { threshold: 0.12, rootMargin: '0px 0px -40px 0px' });

    els.forEach(el => obs.observe(el));
  })();
</script>
